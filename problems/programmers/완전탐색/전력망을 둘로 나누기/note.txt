연결 하나를 끊는다 -> wires의 원소들 중 하나를 제외하고 나머지로 graph를 구성

원래 전력망이 하나의 트리였으니 연결 하나를 끊으면 2개의 트리로 나뉠 것이고,
하나의 전력망만 찾으면 나머지 송전탑들은 모두 또 하나의 전력망으로 연결되어 있을 것

wires를 순회하며 한 쪽이 graph에 포함되어 있으면 다른 한 쪽도 추가하고, 아니면 패스
=> 둘 중 하나의 전력망이 구성됨

구성된 전력망의 송전탑 수가 n'이면 나머지 한 전력망의 송전탑 수는 n - n'
두 전력망의 송전탑 수의 차 Math.abs(2*n' - n)의 최솟값을 업데이트

했더니 틀렸다
wires의 각 원소 [v1, v2]에 대해서 2 <= v1 < v2 <= n라고 했지
n < m일 때 n번째 원소의 v1 <= m번째 원소의 v1이라고는 안 했다
순차적으로 확인하면 원래 연결된 전력망이 끊긴 것으로 계산될 수도 있었다

잘라내고 남은 wires 안에 network에 포함된 송전탑이 존재하는 조건으로 반복 수행해서 해결
