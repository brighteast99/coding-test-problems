주어진 배열 내에서 cycle들을 찾아야 하는 문제
반복 조건만 적절히 설정하면 되는 문제로 판단

원본 배열에서 원소를 삭제할 경우 cycle 계산을 위해 인덱스를 조정해야 함
=> 원본 배열은 그대로 두고 방문 여부를 별도로 체크

1. 확인하지 않은 상자들 중 첫 번째를 확인
2. 해당 위치에서 시작하여 cycle을 확인
    2-1. cycle 크기 증가
    2-2. 해당 위치 방문 여부 체크
    2-3. 카드 값 확인하여 해당 위치 상자 확인
    2-4. 해당 위치 상자를 처음 확인한다면 2-1부터 반복
    2-5. 이미 확인한 상자라면 3으로
3. cycle 크기를 저장
4. 확인하지 않은 상자가 남아있다면 1부터 반복
5. 상자를 모두 확인했다면 종료
